# WireDrop Lite Releases

**WireDrop Lite** is a lightweight, portable Windows companion app for **WireDrop**, designed for fast local network messaging and file transfer without installation.

WireDrop Lite works with the main WireDrop app on Android and Windows, allowing devices on the same Wi-Fi or local network to discover each other and exchange messages, files, and folders quickly.

## Features

- Portable Windows app
- No installation required
- Less than 400 KB
- Works with WireDrop Android app
- Works with WireDrop Windows app
- Local Wi-Fi / LAN based discovery
- One-to-one messaging
- File transfer support
- Folder transfer support
- Lightweight and fast startup

## How to Use

1. Download `WireDropLite.zip` from the latest release.
2. Extract the ZIP file.
3. Run `WireDropLite.exe`.
4. Make sure your devices are connected to the same Wi-Fi or local network.
5. Open WireDrop or WireDrop Lite on the other device.
6. Start sending messages, files, or folders.

## Notes

- WireDrop Lite is intended for quick usage without installing the full Windows app.
- Both devices must be connected to the same local network.
- Some routers may block device discovery due to client isolation, firewall, or multicast restrictions.
- Linux and macOS builds are currently under development.

## About WireDrop

WireDrop is a local network messenger and file transfer application that helps users share messages, files, and folders between devices on the same Wi-Fi or LAN network.

It is designed as a simple cross-platform alternative for local device-to-device communication.
